# TaigerDesktop

Десктопное приложение на WPF для управления пользователями, фотографиями, статистикой и административными задачами.

## Описание

TaigerDesktop — внутренний инструмент компании для:
- Просмотра и управления пользователями
- Проверки загруженных фотографий
- Ведения ежедневной статистики и KPI по автомобилям
- Управления правами администраторов

## Основные возможности

- Авторизация пользователей
- Главная панель с обзором
- Просмотр и поиск пользователей
- Проверка загруженных фотографий (с полноэкранным просмотром)
- Добавление и управление администраторами
- Ежедневная статистика и KPI по автомобилям

## Требования

- Windows 10/11
- .NET 6.0 или выше (в зависимости от настроек проекта)
- Visual Studio 2022+ с workload «Разработка классических приложений .NET»

## Сборка и запуск

1. Откройте `TaigerDesktop.sln` в Visual Studio
2. Восстановите NuGet-пакеты (если используются)
3. Нажмите `F5` для запуска в режиме отладки
